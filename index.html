All files are ready to start assembling your SENS_HAND v1 Dexterous Hand

Important Notice

The current SENS_HAND v1 release does not include sensors as we are still improving the reliability of the sensor version. The sensor version will be released later this year.

Before You Begin

Before starting assembly, make sure you have all necessary tools and parts ready:

  • Complete SENS_HAND v1 parts kit
  • Screwdriver set
  • Pliers
  • Tweezers
  • PTFE tubes and tendon wires

It's recommended to work on an anti-static mat and keep your workspace clean.
